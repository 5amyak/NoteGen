## Backend of NoteGen

Following directory contains the backend of our extension NoteGen. It contains :-
* _uploads_ : this directory contains audio files uploaded by users.
* audioToText.js: server code is divided into modules for better debugging. This file contains code used to get text from audio file.
* translator.js: This file is used to translate code in various languages
* server.js: Most important file, contains all the api used to summarize, convert, translate etc.
